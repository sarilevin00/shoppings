# RTK React Shop - מסך ראשון (קליינט בלבד)

פרויקט פשוט לדמו של מסך רכישה עם React + Redux Toolkit.

מה קיים:
- טענת נתונים (קטגוריות + מוצרים) מ־public/data.json בזמן עליית הדף.
- בחירת קטגוריה והצגת המוצרים שלה.
- שינוי כמות והוספה לסל (השינויים נשמרים ב־Redux).
- תצוגת עגלה בצד עם עריכה/הסרה וסכום סופי.

הרצה:
1. npm install
2. npm start
